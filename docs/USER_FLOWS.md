# User Flow: The "Master-to-Multi" Workflow

## Flow 1: The First Upload
1. **Trigger:** User lands on Dashboard and clicks "New Master Asset".
2. **Action:** User uploads a 30-minute YouTube video link or file.
3. **Processing:** 
    - System transcribes content (Whisper).
    - AI analyzes for "Viral Hooks".
    - System generates 5 Short-form clips and 1 Blog draft.
4. **Review:** User enters "AI Workspace" to review clips.
5. **Branding:** User applies "Brand Identity" (One-click theme application).
6. **Distribution:** User selects "Schedule to TikTok & Reels" and confirms.

## Flow 2: Setting up Brand Voice
1. **Trigger:** User navigates to "Brand Center".
2. **Action:** User uploads a document or provides a URL to an existing blog.
3. **AI Analysis:** System analyzes writing style, emoji usage, and sentence structure.
4. **Validation:** System generates a "Sample Post" for the user to approve.
5. **Outcome:** Every future AI output follows this specific persona.
