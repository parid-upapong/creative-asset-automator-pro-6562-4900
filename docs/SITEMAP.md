# Project OVERLORD: Sitemap v1.0

## 1. Public Facing (Unauthenticated)
- **Home:** Hero section, Value Prop, Features.
- **Pricing:** Tiered subscription plans (Free/Pro/Agency).
- **Resources:** Blog, Case Studies, AI Content Guide.
- **Login / Sign-up:** Clerk/Supabase Auth integration.

## 2. User Dashboard (Authenticated)
- **Dashboard Overview:** Recent activity, storage usage, and quick-start buttons.
- **Asset Library:**
    - `All Assets`: Grid/List view of uploaded media.
    - `Folders/Tags`: Organization system.
    - `Asset Detail View`: Individual file management and "Source" history.
- **AI Workspace:**
    - `Clipping Studio`: Vertical video editor & AI clip selector.
    - `Copywriter`: AI-generated threads, blogs, and captions.
    - `Review Station`: Previewing assets before export.
- **Brand Center:**
    - `Voice Profiles`: Custom instructions for the AI tone.
    - `Style Kits`: Fonts, colors, logos, and watermarks.
- **Publishing Queue:**
    - `Calendar`: Visual view of scheduled posts.
    - `Integrations`: Connect TikTok, YouTube, IG, LinkedIn.
- **Settings:**
    - `Account`: Profile and security.
    - `Billing`: Subscription and invoices.
    - `API Access`: For Pro/Agency developers.

## 3. Operational Pages
- **404 Page:** Custom branded error page.
- **Onboarding Flow:** 3-step wizard to set up first Brand Profile.
- **Success/State Pages:** Payment success, Processing complete notifications.
